How it works:

Click any link — LinkGuard intercepts it and asks if you want to scan it first. Hovering a link pre-scans it locally for speed.

Scan pipeline:
1. In-memory cache
2. URLhaus (malware URLs)
3. OpenPhish (phishing)
4. ThreatFox (threat IOCs)
5. India scam heuristics
6. VirusTotal (requires API key)

Verdicts:
✅ Safe   ⚠️ Suspicious   ⛔ Dangerous   ❓ Unknown
